# create-ts-cronjob

This project is still in development.

[Link to npm](https://www.npmjs.com/package/create-ts-cronjob)

## Getting Started
Run `npx` 
```bash
  npx create-ts-cronjob
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

## Dependencies
